## 一、项目概述

### 1.1 项目背景

本系统是一款面向中小型企业的合同管理平台，实现合同的电子化管理，支持合同全生命周期的跟踪，包括合同创建、收付款登记、发票管理、到期提醒等核心功能。

### 1.2 项目目标

- 实现合同信息的集中统一管理
- 跟踪合同收付款进度，控制财务风险
- 提供发票管理，确保财务凭证完整
- 到期自动提醒，避免合同漏管
- 支持多角色权限管理，满足企业组织架构需求
- 提供数据导出功能，便于财务审计

## 三、角色权限体系

### 3.1 系统角色

| 角色 | 说明 |
|------|------|
| super（超级管理员） | 拥有全部权限，可管理所有数据和用户 |
| normal（普通管理员） | 按权限配置访问功能，可管理业务员 |
| sales（业务员） | 只能操作自己创建的合同数据 |

### 3.2 权限矩阵

| 功能 | 超级管理员 | 普通管理员 | 业务员 |
|------|:----------:|:----------:|:------:|
| 所有合同操作 | ✓ | 按权限 | 按权限 |
| 用户管理 | ✓ | 仅业务员 | ✗ |
| 系统设置 | ✓ | ✗ | ✗ |
| 合同转交 | ✓ | ✗ | ✗ |
| 删除合同 | ✓ | ✗ | ✗ |
| 删除归档合同 | ✓ | ✗ | ✗ |
| 数据隔离 | 不受限制 | 受限 | 受限 |

### 3.3 权限点清单（40项）

| 权限组 | 权限点 |
|--------|--------|

| 收款-进度 | receipt.progress.view / entry / undo / export |
| 收款-记录 | receipt.records.view / export |
| 收款-发票 | receipt.invoices.view / entry |
| 付款-进度 | payment.progress.view / entry / undo / export |
| 付款-记录 | payment.records.view / export |

### 3.4 权限模板

| 模板 | 说明 |

## 四、功能模块详细分析

### 4.1 登录认证模块

| 功能 | 说明 |
|------|------|
| 用户登录 | 用户名+密码认证 |
| CSRF保护 | 所有POST请求需通过csrf_verify验证 |
| 会话管理 | session_regenerate_id防会话固定攻击 |
| 记住密码 | localStorage存储（仅本地模式） |
| 登出 | 清除所有会话数据 |

### 4.2 首页仪表盘

| 功能 | 说明 |
|------|------|
| 合同统计 | 合同总数、收款合同数、付款合同数、各状态数量 |
| 财务统计 | 待收款/待付款/已收款/已付款金额 |
| 趋势图表 | 近7日合同数量和金额折线图（ECharts） |
| 数据隔离 | own_contract_only启用时，非超级管理员只看自己的合同 |

### 4.3 合同管理模块

#### 4.3.1 合同列表

| 功能 | 说明 |
|------|------|
| 列表展示 | 合同编号、名称、类型、款项类型、签约方、金额、已登记金额、已开票金额、待开票金额、截止日期、剩余天数、状态、创建人 |
| 多条件筛选 | 关键词、状态、合同类型、金额达标情况、款项类型（收款/付款） |
| 分页 | 每页20条 |
| 排序 | 按ID倒序 |

#### 4.3.2 新增/编辑合同

| 功能 | 说明 |
|------|------|
| 基本信息 | 合同编号、合同名称、客户名称、签约方、签约人、联系电话 |
| 款项类型 | 收款/付款（有登记记录后锁定不可修改） |
| 合同金额 | 数字输入+金额大写自动转换 |
| 日期信息 | 签订日期、生效日期、截止日期（支持"长期有效"选项） |
| 业务信息 | 合同状态（进行中/已完成/已终止/即将到期）、合同类型 |
| 附件上传 | 支持PDF/Word/图片（jpg/png/gif/webp），多文件上传 |
| 附件管理 | 查看预览、下载、删除已上传的附件 |

#### 4.3.3 合同详情

| 功能 | 说明 |
|------|------|
| 信息展示 | 完整合同信息，含金额大写 |
| 收付款进度 | 进度卡片（合同金额、已登记金额、待登记金额、完成率） |
| 收付款时间线 | 展示所有收付款登记记录 |
| 发票进度 | 进度卡片（可开票金额、已开票金额、待开票金额、开票率） |
| 发票时间线 | 展示所有发票记录 |

#### 4.3.4 合同操作权限

| 操作 | 权限 | 说明 |
|------|------|------|
| 新增合同 | contracts.create | |
| 编辑合同 | contracts.edit | 已完成合同不允许编辑 |
| 删除合同 | contracts.delete | 仅超级管理员可删除 |
| 更新状态 | contracts.update_status | 仅能设为"已完成"或"已终止" |
| 归档合同 | contracts.archive | 仅已完成合同可归档 |
| 合同转交 | 仅超级管理员 | 将合同创建人转移给其他业务员 |

### 4.4 收款/付款业务模块

#### 4.4.1 收付款进度

| 功能 | 说明 |
|------|------|
| 进度查看 | 合同列表，展示每合同的总额、已登记金额、剩余金额、进度条 |
| 登记收款/付款 | 弹窗选择合同，输入金额、备注、上传凭证（支持PDF/图片） |
| 撤销上一次 | 仅能撤销最新一条记录，且每个合同只能撤销一次 |
| 导出 | 导出CSV格式的进度报表 |

#### 4.4.2 收付款记录

| 功能 | 说明 |
|------|------|
| 记录列表 | 时间、合同、登记人、合同金额、本次金额、登记后剩余金额、备注、凭证 |
| 凭证查看 | 可在线查看或下载凭证文件 |
| 导出 | 导出CSV格式 |

#### 4.4.3 发票管理

| 功能 | 说明 |
|------|------|
| 开票明细 | 合同维度的开票情况（已登记金额、已开票金额、待开票金额） |
| 开票记录 | 时间、合同、开票金额、登记人、备注、附件 |
| 上传发票 | 弹窗选择合同，输入开票金额、备注、上传发票附件 |
| 业务校验 | 开票金额不能大于已登记收付款金额 |
| 导出 | 导出CSV格式 |

### 4.5 归档合同模块

| 功能 | 说明 |
|------|------|
| 归档列表 | 展示所有已归档合同 |
| 筛选 | 关键词、状态、合同类型、款项类型 |
| 导出 | 批量导出归档合同 |
| 删除 | 仅超级管理员可删除归档合同（同时删除附件、凭证、发票文件） |

### 4.6 用户管理模块

### 4.7 系统设置模块

#### 4.7.1 应用设置

## 六、安全特性

| 安全措施 | 说明 |
|----------|------|
| CSRF防护 | 所有POST表单包含csrf_token，服务器端验证 |
| 密码加密 | password_hash() BCRYPT算法 |
| SQL注入防护 | PDO预处理语句 |
| XSS防护 | htmlspecialchars()输出转义 |
| 文件上传安全 | MIME类型白名单校验 |
| 权限细粒度控制 | 40项独立权限点 |
| 数据隔离 | own_contract_only可配置业务员只能操作自己的合同 |

系统支持将数字金额自动转换为中文大写，例如：
- 123456.78 → 壹拾贰万叁仟肆佰伍拾陆元柒角捌分

| 状态 | 说明 |

| 类型 | 说明 |
|------|------|
