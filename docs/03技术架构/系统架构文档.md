## 一、系统架构概览

### 1.1 架构类型

本系统采用**混合型架构**（非标准MVC），基于原生PHP实现，无框架依赖。


### 3.2 路由机制

本系统**无框架路由**，采用**基于文件名直接映射**的简单路由：

| URL路径 | 映射文件 | 参数 |
|---------|----------|------|
| `/` | index.php | 无 |
| `/login.php` | login.php | 无 |
| `/orders.php` | orders.php | `?page=&kw=&status=` |
| `/contract_view.php?id=1` | contract_view.php | `$_GET['id']` |
| `/contract_form.php?id=1` | contract_form.php | `$_GET['id']` (编辑模式) |
| `/api/admin_account.php` | api/admin_account.php | JSON Body |

## 四、核心模块架构

### 4.1 引导程序 (bootstrap.php)

```php
// 初始化流程伪代码
function init() {
    // 1. 安装检查
    $lockFile = $root . '/data/installed.lock';
    if (!is_file($lockFile) && $script !== 'install.php') {
        header('Location: /install.php');
        exit;
    }

    // 2. 配置加载
    $cfg = require __DIR__ . '/config/config.php';
    date_default_timezone_set($cfg['app']['timezone']);

    // 3. 会话初始化
    if (session_status() !== PHP_SESSION_ACTIVE) {
        session_start();
    }

    // 4. 核心库加载
    require_once __DIR__ . '/functions.php';
    require_once __DIR__ . '/auth.php';
    require_once $root . '/config/database.php';
    require_once __DIR__ . '/site_branding.php';

    // 5. Schema初始化
    mf_ensure_contract_schema(db());
}
```

### 4.2 数据库层 (database.php)

**PDO单例模式**：

```php
function db(): PDO {
    static $pdo = null;
    if ($pdo instanceof PDO) {
        return $pdo;
    }
    $cfg = require __DIR__ . '/config.php';
    $dsn = sprintf(
        'mysql:host=%s;port=%d;dbname=%s;charset=%s',
        $cfg['db']['host'],
        (int) $cfg['db']['port'],
        $cfg['db']['name'],
        $cfg['db']['charset']
    );
    $pdo = new PDO($dsn, $cfg['db']['user'], $cfg['db']['pass'], [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
        PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
        PDO::ATTR_EMULATE_PREPARES => false,  // 强制服务端预处理
    ]);
    return $pdo;
}
```

### 4.3 认证授权 (auth.php)

**三层安全检查**：

```
┌─────────────────────────────────────────────────┐
│              第一层：登录检查                      │
│         require_login() - 会话有效性              │
│    检查：$_SESSION['admin_id'] 是否存在           │
└───────────────────────┬─────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────┐
│              第二层：权限检查                      │
│         admin_can('permission.key')              │
│    检查：权限点是否在 $_SESSION['permissions']    │
│    super角色拥有所有权限 (*)                      │
└───────────────────────┬─────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────┐
│              第三层：数据隔离                      │
│    mf_own_contract_only_enabled()               │
│    检查：own_contract_only 配置是否启用           │
│    启用后：非super只能操作自己的合同              │
└─────────────────────────────────────────────────┘
```

### 4.4 页面渲染 (layout.php)

**Buffer + Include 模式**：

```php
// 页面文件 (orders.php)

?>
<div class="page-header">
    <h1>合同列表</h1>
</div>
<table class="table">...</table>
<?php
$content = ob_get_clean();

// 渲染布局

// layout.php 结构
// ┌────────────────────────────────────┐
// │  HTML头部 + 资源引用 (CSS/JS)        │
// ├────────────────────────────────────┤
// │  顶部栏 (Logo + 用户信息)           │
// ├──────────┬─────────────────────────┤
// │  侧边栏   │  $content              │
// │ (权限菜单)│  (页面内容)              │
// ├──────────┴─────────────────────────┤
// │  页脚 (JS脚本)                      │
// └────────────────────────────────────┘
```

### 4.5 核心函数库 (functions.php)

| 函数分类 | 关键函数 |
|----------|----------|
| 工具函数 | `e()` - XSS转义, `redirect()` - 重定向, `url()` - URL生成 |
| 数据库 | `db()` - PDO单例 |
| CSRF | `csrf_token()` - 生成令牌, `csrf_verify()` - 验证令牌 |
| 权限 | `admin_can()` - 权限检查, `require_permission()` - 权限强制 |
| 权限定义 | `mf_permission_catalog()` - 40项权限清单, `mf_permission_templates()` - 权限模板 |
| Schema | `mf_ensure_contract_schema()` - 自动建表/升级 |
| UI组件 | `mf_contract_status_*()` - 状态标签生成 |

### 7.2 CSRF防护机制

```php
// 令牌生成 (functions.php)
function csrf_token(): string {
    if (empty($_SESSION['csrf_token'])) {
        $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
    }
    return $_SESSION['csrf_token'];
}

// 令牌验证 (functions.php)
function csrf_verify(?string $token): bool {
    return is_string($token)
        && isset($_SESSION['csrf_token'])
        && hash_equals($_SESSION['csrf_token'], $token);
}

// 使用方式：表单中嵌入
<input type="hidden" name="csrf" value="<?= e(csrf_token()) ?>">
```
